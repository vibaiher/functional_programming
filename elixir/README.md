Installing Phoenix
------------------

### Install nodejs

```
curl -sL https://deb.nodesource.com/setup_5.x | sudo -E bash -
sudo apt-get install nodejs
```

### Install hex

```
mix local.hex
```

### Install Phoenix

```
mix archive.install https://github.com/phoenixframework/phoenix/releases/download/v1.1.1/phoenix_new-1.1.1.ez
```
